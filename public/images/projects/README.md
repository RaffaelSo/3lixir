# Project Image Structure

Store all project images here, grouped by project slug.

## Folder Layout

- `cyber-forest/` (Cyber Forest, 2024)
- `cyber-wasteland/` (Cyber Wasteland, 2025)
- `arctica-3d/` (Arctica 3D, 2025)
- `artica/` (Artica, 2025)
- `lookbook/`

## Naming Convention

Use predictable file names per project:

- `hero.jpg` (main card/detail hero image)
- `01.jpg`, `02.jpg`, `03.jpg` (editorial sequence)

If needed, use format-specific extensions (`.webp`, `.png`) but keep the same base names.

## Data Mapping

Reference files from `src/data/projects.ts` with root-relative paths, for example:

- `/images/projects/nocturne-index/hero.jpg`
- `/images/projects/nocturne-index/01.jpg`
